Quick start

Three ways to import. Pick the one that fits your project.

CDN — two files

<link rel="stylesheet" href="sprout.min.css">
<script src="sprout.min.js" defer></script>

Single script drop-in

<script src="sprout.bundle.js" defer></script>

npm

npm install sprout-ui

import 'sprout-ui/dist/sprout.min.css';
import Sprout from 'sprout-ui';

Toast Sprout.toast()

JS API only — no HTML markup needed. Sprout injects the container automatically. Stacks, auto-dismisses, and supports position options.

Preview — click to trigger

Code

// Default
Sprout.toast('File saved.');

// With type
Sprout.toast('Something went wrong.', {
  type: 'danger'
});

// With duration + position
Sprout.toast('Update available.', {
  type: 'info',
  duration: 5000,
  position: 'top-right'
});

// Persistent (duration: 0)
Sprout.toast('Action needed.', {
  type: 'warning',
  duration: 0
});

Preview

What is Sprout?

Sprout is an ultra-lightweight, sustainable, zero-dependency UI library. Drop in two files and you have a complete UI system.

How do I customise themes?

Override any --sp-* CSS variable. Set data-theme="dark|light|high-contrast" on <html>, or omit it to follow the OS preference automatically.

Does it work without JavaScript?

Most components are pure CSS. Only Tabs, Modal, Dropdown, Toast, and the theme switcher API require JS.

<!-- Open by default -->
<details open>
  <summary>Question</summary>
  <p>Answer goes here.</p>
</details>

<!-- Collapsed -->
<details>
  <summary>Another question</summary>
  <p>Another answer.</p>
</details>
