# Layout

Sprout includes four layout primitives: **container**, **grid**, **stack**, and **cluster**.  
These utilities help structure content without adding unnecessary complexity.

For a high-level explanation and a combined demo, see [Layout overview](/foundation/layout-overview).

## Detailed pages

- [Container](/layout/container) — constrains width and adds horizontal padding
- [Grid](/layout/grid) — creates responsive column layouts
- [Stack](/layout/stack) — adds consistent vertical spacing between items
- [Cluster](/layout/cluster) — groups items horizontally with wrapping support
