# Typography

Sprout styles semantic text elements (headings, paragraph, code, blockquote, etc.) directly. The full typography guide lives in Foundation.

**See [Typography](/foundation/typography)** for headings (h1–h6), paragraph, small, strong, code, pre, kbd, blockquote, hr, and a live demo.

For token reference (font sizes, families, line height), see [Tokens → Typography](/tokens/typography).
